# openshift-simple-minio-deployment
Repo to store the minimal required yaml files to deploy minio at an OCP cluster

# TL;DR;

````bash
bash bootstrap.sh
````

# Deployed resources

- MinIO namespace
- PVC
- Service
- WebUI route
- API route
- MinIO k8s deployment
